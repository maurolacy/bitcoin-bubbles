# Extra / Experimental Code

This folder contains experimental notebooks and scripts used during research and development.

## Contents

- **best_aicc.R** - R script for AICc optimization.
- **loess_aic.R** - R script for LOESS AIC calculations.
- **loess.r** - R script for LOESS smoothing.
- **loessSpan.nb** - Mathematica notebook for LOESS span analysis.
- **ProfileLikelihood.nb** - Profile likelihood analysis.
- **BreuschPagan.nb** - Breusch-Pagan test implementation.
- **ethereumUsers.nb** - Experimental Ethereum users analysis.
- **Nonparametrix.m Intro - Bernard Gress - 2004.nb** - Reference material on nonparametric methods.
- **Bitcoin_2030.jpg** - Reference image.

These files are provided for reference and may not be directly related to the main implementation. They represent exploratory work and research experiments.
